Curious Drinker

A data visualization app that shows Foursquare’s most popular bars in the Lower East Side and the gender ratio of their visitors. Curious Drinker uses Foursquare's API to pull realtime data daily, and provides users with visualization of the data.

Utilized Backbone.js with the Rails Asset Pipeline and integrated D3.js and Google Maps API to display data on the correct map coordinates.

Screen Test

A personality test game that analyzes the user based on the plots of the user’s choice of movies. Screen Test combines Chart.js and the text analytics API "The Values Classifier" by PRfekt, to provide users with a visualization of their personality test results.

Built a custom MVC framework similar to Backbone.js utilizing Underscore.js.

Second Date

A platform for users to act as an online wingman/wingwoman by sharing opinions on venues and categorizing them by voting on whether the specific venue is good for first, second, third date, or just plain bad for any dates. Second Date queries Foursquare's API to get the most popular date spots according to tips and recommendations made by Foursquare users.

Implemented methods to keep track of votes and comments made by users through multiple join tables.
